Cisco 2500

The Cisco 2500 series router provides a wide range of routing capabilities primarily in a LAN or branch office environment.

The lab has the following 2500 routers:

(23) 2501 - Single LAN ethernet router
( 9 ) 2503 - Access router with ISDN
( 2 ) 2504 - Single LAN token ring router
( 3 ) 2509 - Access Server
( 2 ) 2511 - Access Server
( 2 ) 2514 - Dual LAN ethernet router
( 1 ) 2523 - High density serial router
( 2 ) 2524 - Modular WAN router

Cisco 2600

The Cisco 2600 is a modular router offering versitality and integration capabilities which is not provided in the 2500 series routers.

The lab has the following 2600 routers:

(11) 2620 - Modular router with ethernet and serial ports.

  
 

Cisco 3600

The Cisco 3600 series is a family of modular, multiservice access routers for medium to large offices or smaller ISPs.

The lab has the following 3600 routers:

(1) 3620 - 2 slot modular router with ethernet and serial.
(2) 3640 - 4 slot modular router with voice, serial, and fast ethernet

Cisco 4000

The Cisco 4000 series is a modular router offering a wide range of connectivity features and powerful internetworking software.

The lab has the following 4000 routers:

(2) 4000 - 2 slot modular router with serial and ISDN.

Because of the 4000's flexibility, it serves as a perfect platform for learning and experimentation.

 

  
 

Cisco 7000

The Cisco 7000 series is a high speed multiprotocol router. It is primarily used as a edge or core router.

The lab has the following 7000 routers:

(6) 7000 - 7 slot router with ethernet, serial and ATM
(2) 7500 - 7 slot router with fast ethernet and serial.

 

Cisco 7200

The Cisco 7200 series is a high speed modular router offering a wide range of scalable capabilities.

The lab has the following 7200 routers:

(1) 7204 VXR - 4 slot router with fast ethernet and ATM.

The 7200 comes equipped with Cisco's fastest processor, which can satisfy the most demanding performance requirements.
